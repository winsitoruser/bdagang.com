================================================================================
API ENDPOINTS SUMMARY
================================================================================

GET /api/hq/requisitions/available-branches?requisitionId={id}
Purpose: Get list of branches that can fulfill requisition
Auth: Required (session)
Response: Array of branches with id, code, name, type, city, stock, distance

PUT /api/hq/requisitions/{id}/assign-fulfillment
Purpose: Assign fulfilling branch to requisition
Auth: Required (session)
Body: { fulfillingBranchId, estimatedDeliveryDate, notes }
Response: Updated requisition with associations

POST /api/hq/requisitions/export
Purpose: Export requisitions data
Auth: Required (session)
Body: { format: 'csv'|'json', filters: {...} }
Response: File download (CSV or JSON)
