API ENDPOINTS NEEDED
================================================================================

NEW ENDPOINTS REQUIRED:

1. GET /api/fleet/drivers/available
   Purpose: Get available drivers for assignment
   Filters: branch, license_type, availability_status
   Returns: List of available drivers with details

2. GET /api/fleet/vehicles/available
   Purpose: Get available vehicles for assignment
   Filters: branch, vehicle_type, status, capacity
   Returns: List of available vehicles with details

3. POST /api/fleet/assignments
   Purpose: Assign vehicle and driver to shipment
   Body: { shipmentId, vehicleId, driverId }
   Actions:
   - Update shipment with vehicle_id, driver_id
   - Update vehicle status to "on_route"
   - Update driver availability to "on_duty"
   - Create assignment record

4. PUT /api/fleet/assignments/:id/complete
   Purpose: Complete assignment and update metrics
   Body: { actualOdometer, deliveryStatus }
   Actions:
   - Update driver performance
   - Update vehicle odometer
   - Mark resources as available
   - Calculate metrics

5. GET /api/fleet/vehicles/:id/assignments
   Purpose: Get assignment history for vehicle
   Returns: List of shipments assigned to vehicle

6. GET /api/fleet/drivers/:id/assignments
   Purpose: Get assignment history for driver
   Returns: List of shipments assigned to driver
