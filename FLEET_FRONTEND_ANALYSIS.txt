COMPONENT SPECIFICATIONS
================================================================================

1. VEHICLE FORM MODAL
Purpose: Add new vehicle or edit existing
Fields:
- License Plate* (required)
- Vehicle Type* (dropdown: truck, van, motorcycle, car)
- Brand
- Model
- Year
- Color
- Ownership Type* (dropdown: owned, leased, rental)
- Purchase Date (if owned)
- Purchase Price (if owned)
- Lease Details (if leased)
- Max Weight (kg)
- Max Volume (m³)
- Fuel Tank Capacity (L)
- Registration Number (STNK)
- Registration Expiry
- Insurance Policy Number
- Insurance Provider
- Insurance Expiry
- Assigned Branch (dropdown)
- Notes

Validation:
- License plate required & unique
- Vehicle type required
- Ownership type required

Actions:
- Save (POST /api/fleet/vehicles or PUT /api/fleet/vehicles/:id)
- Cancel

2. DRIVER FORM MODAL
Purpose: Add new driver or edit existing
Fields:
- Full Name* (required)
- Phone*
- Email
- Address
- Date of Birth
- License Number* (required)
- License Type* (dropdown: SIM A, SIM B1, SIM B2, SIM C)
- License Issue Date
- License Expiry Date
- Employment Type (dropdown: permanent, contract, freelance)
- Hire Date
- Assigned Branch (dropdown)
- Notes

Validation:
- Full name required
- License number required & unique
- License type required
- Phone required

Actions:
- Save (POST /api/fleet/drivers or PUT /api/fleet/drivers/:id)
- Cancel

3. SEARCH FUNCTIONALITY
Implementation:
- Real-time search as user types
- Search in: license plate, vehicle number, brand, model (vehicles)
- Search in: name, driver number, phone (drivers)
- Debounce 300ms
- Clear search button

4. FILTER FUNCTIONALITY
Vehicles Filters:
- Status (all, active, maintenance, retired)
- Type (all, truck, van, motorcycle, car)
- Branch (all, branch1, branch2, etc)
- Ownership (all, owned, leased, rental)

Drivers Filters:
- Status (all, active, on_leave, suspended)
- Availability (all, available, on_duty, off_duty)
- License Type (all, SIM A, SIM B1, SIM C)
- Branch (all, branch1, branch2, etc)

UI: Dropdown panel with checkboxes/radio buttons

5. EXPORT FUNCTIONALITY
Formats: CSV, Excel
Data: Current filtered/searched data
Columns: All visible columns + additional details
Button: Top right with download icon

6. VEHICLE DETAIL MODAL
Sections:
- Basic Info (plate, type, brand, model, year, color)
- Ownership Info (type, purchase/lease details)
- Capacity (weight, volume, fuel tank)
- Documentation (registration, insurance)
- Assignment (branch, driver)
- Service History (last service, next due)
- Trip History (if available)
- Notes

Actions:
- Edit (opens VehicleFormModal)
- Delete
- Close

7. DRIVER DETAIL MODAL
Sections:
- Personal Info (name, phone, email, address, DOB)
- License Info (number, type, issue/expiry dates)
- Employment Info (type, hire date, branch)
- Performance Metrics (deliveries, on-time rate, safety score, rating)
- Delivery History (if available)
- Notes

Actions:
- Edit (opens DriverFormModal)
- Delete
- Close
