API FUNCTIONALITY
================================================================================

VEHICLES API (/api/fleet/vehicles):

GET - List Vehicles:
- Filter by status (active, maintenance, retired)
- Filter by type (truck, van, motorcycle, car)
- Filter by branch
- Returns stats (total, active, maintenance, on route)

POST - Create Vehicle:
- Validates required fields (license plate, type)
- Checks duplicate license plate
- Generates vehicle number
- Returns created vehicle

Stats Returned:
{
  total: 5,
  active: 4,
  maintenance: 1,
  onRoute: 4
}

DRIVERS API (/api/fleet/drivers):

GET - List Drivers:
- Filter by status (active, on_leave, suspended)
- Filter by availability (available, on_duty, off_duty)
- Filter by branch
- Returns stats (total, active, available, on duty)

POST - Create Driver:
- Validates required fields (name, license)
- Checks duplicate license number
- Generates driver number
- Returns created driver

Stats Returned:
{
  total: 5,
  active: 5,
  available: 2,
